Open decisions log

Unresolved design questions. Include this page in LLM sessions when the task touches a listed topic. Update status as you go—prevents re-arguing the same call every few weeks.

Status key

  • Open — not settled; gather options.
  • In discussion — leaning exists; needs final sign-off.
  • Locked — canon; only change via changelog + stakeholder note.

How to use

  1. Add a row when a question appears—one sentence topic + notes column.
  2. Link out to STORY / LEVEL_DESIGN / etc. when a doc holds detail.
  3. When locking, move the rationale into changelog and set status to Locked.
