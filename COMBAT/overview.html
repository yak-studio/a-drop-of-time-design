Combat overview

Tone, Fracture Charge, Teleattack, and how swap ties to marriage theme.

Combat tone rule

Heavy, grounded, uncomfortable. No flashy heroism. Hits are decisions, not celebration. If combat reads “fun-first,” story collapses.

Escalation (Act I template)

Encounters evolve: confusion → resistance → intimacy (first Temporal) → moral pressure → organised defence. If two fights feel identical, cut or mutate one.

Emotional integration

Narrative is not layered on after — fights create guilt; memories contextualise guilt; choices weaponise guilt.

Fracture Charge & forced swap

Shared resource builds in sustained combat. When full, mandatory swap — mid-animation, mid-risk. Player reads meter and sets up handoffs; cannot cancel the swap.

Mother: faster, aggressive. Father: slower, deliberate; high payoff windows. Enemies differ by plane.

Signature fight: Act III dual-Temporal (one per plane) — swap becomes tool to interrupt charge cycles. See encounter philosophy.

Teleattack

Introduced Act I after first Temporal kill. Input after attack chain: swap to other parent continuing combo; disorienting camera ok. Bridges navigation swap to weaponised fracture. Stamina split across parents extends window.

Memory extraction

Only on Temporal kills. Parents blank; player bears weight. Feeds “Coalition was people.”
